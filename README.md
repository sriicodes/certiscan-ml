# Machine Learning Mini Project

This repository contains a machine learning project originally developed in a Kaggle notebook.

## Project Structure

```text
project/
│
├── notebooks/
│   └── mini-project.ipynb
│
├── src/
│   ├── preprocess.py
│   ├── train.py
│   └── predict.py
│
├── requirements.txt
├── README.md
└── .gitignore
```

## Features

- Data preprocessing
- Machine learning workflow
- Model training and evaluation
- Kaggle notebook compatible

## Libraries Used

- albumentations
- matplotlib
- numpy
- opencv-python
- os
- pandas
- scikit-learn
- segmentation_models_pytorch
- torch
- tqdm

## Installation

```bash
pip install -r requirements.txt
```

## Running the Notebook

```bash
jupyter notebook
```
